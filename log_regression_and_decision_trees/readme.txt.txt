since I used the argparse-module in ex1.py, there is an argument "--datadir" which takes in the path to the data folder 
where the training/testing diabetis data is stored in.


So to run the script via command line it will look like:

"python ex1.py --datadir <your_path>"

(e.g. in my case: "python ex1.py --datadir data") 

-------------------------------------------------------------------------------

The ex2.py script runs without any argparse arguments, i.e.

"python ex2.py"
